# Baden Appleyard Health Dashboard

Personal health monitoring dashboard — deployed via GitHub Pages.

## Live URL
**https://badapple71-hash.github.io/health-dashboard/**

## Features
- 8 tabs: Overview, Cardiac, Lab Trends, Supplements, Fitness, Trackers, History, Team
- LDL/HDL/glucose/liver enzyme/PSA/TSH trend charts (Chart.js)
- Fitness tracker: steps, resting HR, sleep, VO2 max (manual entry + Google Fit coming)
- Blood pressure log + weight/BMI tracker with persistent localStorage
- Daily supplement & medication reminder checklist
- Light/Dark theme toggle
- PWA-installable on Android

## To Update
Push a new index.html to the main branch — GitHub Pages redeploys automatically.
